# Project Brief

## 1. Project Name

LINGKOD: A Geospatial Intelligence Platform for Disaster Risk Reduction and Management in Camarines Sur.

## 2. Project Goals

To develop a comprehensive situational awareness and decision support system that enhances the disaster management capabilities of the Provincial Government of Camarines Sur.

## 3. Scope

The project will focus on delivering a web-based dashboard that provides real-time data visualization, threat analysis, and communication tools for disaster response teams.

## 4. Target Audience

- Provincial Disaster Risk Reduction and Management Council (PDRRMC)
- Local Government Units (LGUs) in Camarines Sur
- Emergency Responders
- General Public (for receiving alerts and information)

## 5. Key Features

- Interactive map with real-time data overlays
- Real-time data panels for various data sources (e.g., weather, sensor data)
- Insight priority queue for highlighting critical events
- Detailed insight panel for in-depth analysis
- Secure user authentication and role-based access control
